# Análise Comparativa de Algoritmos de Aprendizado de Máquina para Otimização de Rotas em Redes de Comunicação

Este repositório reúne os materiais do Trabalho de Conclusão de Curso (TCC) em Engenharia da Computação, atualmente **em desenvolvimento**, cujo objetivo é analisar e comparar algoritmos de aprendizado de máquina aplicados à otimização de rotas em redes de comunicação simuladas. O projeto contempla fundamentação teórica, metodologia experimental e a implementação de experimentos computacionais para avaliação de desempenho.

## Estrutura do repositório

- `tcc/`: versões parciais e a versão final (quando concluída) do TCC em PDF  
- `experimento/`: código-fonte da simulação e dos experimentos  
- `resultados/`: gráficos, tabelas e evidências geradas a partir dos experimentos  

## Como executar o experimento

### Requisitos
- Python 3.9+  
- Bibliotecas listadas em `requirements.txt`

### Execução

```bash
pip install -r experimento/requirements.txt
python experimento/experimento_tcc.py
